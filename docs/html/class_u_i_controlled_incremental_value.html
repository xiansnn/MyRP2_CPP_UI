Detailed Description

The UIControlledIncrementalValue is a kind of UIModelObject that have special feature such as a value that can be incremented or decremented. This value runs between a min_value and a max_value.

The increment value is configurable. A is_wrappable flag indicates how the value behaves once min or max values are reached.
