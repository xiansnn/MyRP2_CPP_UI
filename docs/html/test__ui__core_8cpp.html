Detailed Description

This is the main test program of the UI_core.

Author
xiansnn (xians.nosp@m.nn@h.nosp@m.otmai.nosp@m.l.co.nosp@m.m)

It consists in simulating 3 controlled incremental values, a display widget on the serial monitor and and a rotary encoder as control device, and a View/Control Manager.

If the manager IS_ACTIVE, rotating the encoder changes the focus, a short click on the encoder central switch, makes IS_ACTIVE the current controlled value with focus. Rotating the encoder changes now the value of the controlledIncrementalValue. A short click, now, validates the value and pass the control to the manager.
