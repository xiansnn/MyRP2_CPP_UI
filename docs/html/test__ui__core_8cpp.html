Detailed Description

This is the main test program of the UI_core.

Author
xiansnn (xians.nosp@m.nn@h.nosp@m.otmai.nosp@m.l.co.nosp@m.m)

It consists in simulating 3 controlled incremental values, a display widget on the serial monitor and and a rotary encoder as control device, and a View/Control Manager.

If the manager IS_ACTIVE, rotating the encoder changes the focus, a short click on the encoder central switch, makes IS_ACTIVE the current controlled value with focus. Rotating the encoder changes now the value of the controlledIncrementalValue. A short click, now, validates the value and pass the control to the manager.

main steps

1- setup serial terminal that uses printf().

2- create 3 incremental value object

3- create 3 serial terminal widget associated with incremental value objects.

4- create a manager connected to the rotary encoder.

5- create a widget for the manager

6- create a set of widget and populate it with all above widgets

7- set focus on the first incremental value object

8- populate manager with managed object

9- start infinite loop

  • sample the rotary encoder central switch
  • give the sampled event to the manager, to let it process the event
  • let the set_of_widget execute draw_refresh
  • sleep for 20ms
