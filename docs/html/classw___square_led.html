void w_SquareLed::draw_refresh ( )
virtual

(re)draw the graphical elements of the widget.

To save running time, we can (re)draw the widget only if the associated UIModelObject has_changed.

Guidance to implement this function:

  • First: Scan all contained sub-widgets if any and call draw_refresh() member function of each of them.
  • Then: apply draw_refresh() on the containing widget if any changes require a screen redraw and finally : clear model change flag if needed. WARNING : When several widget display one Model, only le last one must clear_change_flag()
